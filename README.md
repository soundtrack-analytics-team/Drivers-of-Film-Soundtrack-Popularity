# Drivers of Film Soundtrack Popularity
Khurram Abbas, Tony Lett, Jaime Singson

For our Milestone I project for the University of Michigan's Applied Data Science (MADS) program, we studied the film-level and musical drivers of soundtrack streaming performance. The notebooks in this repository include the full scope of activites that we performed:
  1. Extracting and joining data from multiple APIs (including The Movie Database (TMDB), Spotify, and Last.fm) and a SQL database (MusicBrainz)
  2. Data cleaning, pre-processing, and validation
  3. EDA and visualizations
  4. Feature engineering, correlation analyses, and linear regression

The datasets we include in the repository are samples of the first 1k rows, covering the steps of the pipeline post-join. For a summary of our findings, please see the PDF included in the repo which contains the final report we submitted for a grade.
